# Housing Platform Frontend

Vue.js frontend for the Housing Platform application.

## Features

- Real Estate Company Registration
- Real Estate Agent Registration
- Property Submission
- Public Property Viewing
- User Authentication & Registration

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

The application will be available at http://localhost:3000

## Build

```bash
npm run build
```

## Tech Stack

- Vue 3
- Vue Router
- Pinia (State Management)
- Axios (HTTP Client)
- Tailwind CSS
- Vite (Build Tool)
